## Development

All tasks are in [Drakefile.ts](https://git.io/JvhVf).

### Run all tests

```sh
deno run -A --unstable Drakefile.ts all
```

### Fix files

```sh
env UPDATE=true deno run -A --unstable Drakefile.ts all
```
